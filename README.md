# 🎓 University Management System

A complete, full-stack University Management System built with React, Node.js, and MySQL, fully containerized with Docker for easy deployment.

## 📋 Overview

This University Management System is a comprehensive platform designed to streamline educational institution operations. It provides separate portals for **Administrators**, **Teachers**, and **Students** with role-specific functionalities.

### ✨ Key Features

| Role | Features |
|------|----------|
| **Admin** | Dashboard, User Management, Course Management, Fee Management, Announcements, Schedule Management, Grade Overview |
| **Teacher** | Course Management, Assignment Creation, Grade Submission, Student Management, Schedule View, Announcements |
| **Student** | Course Enrollment, Assignment Submission, Grade View, Fee Payment, Schedule View, Profile Management |

## 🚀 Tech Stack

### Frontend
- **React 18** - UI Library
- **Material-UI (MUI) v5** - Component Library
- **React Query** - Data Fetching & Caching
- **React Router v6** - Navigation
- **Formik & Yup** - Form Validation
- **Recharts** - Data Visualization
- **Socket.io-client** - Real-time Updates

### Backend
- **Node.js** - Runtime Environment
- **Express** - Web Framework
- **Sequelize ORM** - Database Management
- **MySQL** - Database
- **JWT** - Authentication
- **Socket.io** - Real-time Communication
- **Nodemailer** - Email Service
- **Bcrypt** - Password Hashing

### DevOps
- **Docker** & **Docker Compose** - Containerization
- **Git** - Version Control
